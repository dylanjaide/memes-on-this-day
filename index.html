About this site


Memes On This Day is an "on this day"-style calendar, with a focus on the history of internet memes. You can use the form to look up any day of the year, and find an event from meme history which happened on that day in a past year.


This site was made by Dylan Jaide White in March 2021, and is hosted using GitHub Pages. It is currently still a work-in-progress, mostly because more memes are needed to cover all days of the year. So far, we have X memes covering Y out of 366 days. This means that Z days still need to be filled in! If you would like to contribute, you can do so on the GitHub page.
